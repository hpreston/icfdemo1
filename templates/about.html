About this Demo

Current date/time: {{ current_time | datetimefilter }}


This is a simple multi-tier application written in Python to demonstrate the capabilities of Cisco's Intercloud Fabric for building a robust Hybrid Cloud. The presentation layer (web) is hosted in the Public Cloud and accessed directly through the internet to the Cloud Provider. The logic layer (app) is also hosted in the Public Cloud, and is secured from direct internet access by the Intercloud Fabric Policies. The data layer is being hosted in the enteprise private data center and only accessible from the logic layer.
